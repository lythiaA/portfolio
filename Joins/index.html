SQL Quest Hub: Mastering Joins

The Challenge

Instructional Problem

Many technical learners understand the syntax of SQL JOINs but struggle to apply them conceptually. Traditional e-learning often presents this topic passively, leading to low engagement, poor knowledge retention, and a lack of confidence when faced with real-world database problems.

The Learner Persona

The target learner is an adult learner in a Data Management course who is comfortable with basic SQL but needs to build practical, applicable skills. They are motivated by progress and clear goals but can be discouraged by dry, theoretical content. They need a safe environment to practice and make mistakes without consequence.

The Solution & My Process

To solve this, I designed and developed the SQL Quest Hub, a fully interactive, gamified e-learning module built with HTML and CSS. The instructional design is grounded in Gagné's Nine Events of Instruction, ensuring learners are guided from basic concepts to complex application. It also heavily incorporates Keller's ARCS Model of motivation by capturing Attention (gamified story), building Relevance (practical challenges), fostering Confidence (leveled quests), and providing Satisfaction (earning rewards).

Key Features & Skills Demonstrated

Instructional Design

Applied established learning theories (Gagné, ARCS) to structure content, build motivation, and create a logical, scaffolded learning path from simple to complex tasks.

Gamification for Engagement

Integrated game mechanics like points (XP), lives, streaks, and unlockable levels to drive learner motivation and make learning technical content enjoyable.

Front-End Development

Developed a fully functional, single-page application from scratch using modern HTML5 and Tailwind CSS to create a dynamic and responsive experience.

Authentic Assessment

Designed challenge activities that require learners to interact with realistic data tables, testing their ability to apply knowledge rather than just recall syntax.

UI/UX Design

Created a clean, intuitive, and aesthetically pleasing interface with custom backgrounds and a clear visual hierarchy to enhance the learner experience and reduce cognitive load.

Adaptive Learning (Lite)

The final challenge adjusts its difficulty based on learner performance, providing a more challenging experience for advanced users while supporting those who need more practice.

Measuring Success

Evaluation Plan

Learner Engagement:

"How does a gamified, story-driven approach affect a learner's motivation and satisfaction with the training?"

Measured via: A post-module survey asking learners to rate their engagement and the enjoyability of the format on a 5-point Likert scale.

Learning Outcome:

"How does completing the interactive module impact a learner's ability to solve practical SQL JOIN problems?"

Measured via: Analyzing performance data from the final challenge, comparing scores to a pre-assessment, and tracking the number of attempts needed.
